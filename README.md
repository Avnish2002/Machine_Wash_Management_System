# Machine Wash Management System 🚗

A professional web-based platform designed to streamline the scheduling and management of car wash services. This project was developed to provide an intuitive experience for users to book services and for administrators to efficiently manage service requests and status updates.

## 🚀 Key Features

* **User Authentication:** Secure registration and login system for customers.
* **Service Scheduling:** Easy-to-use booking interface to select wash services.
* **Administrative Dashboard:** A dedicated portal for admins to manage service requests, update order statuses, and oversee operations.
* **Data Persistence:** Integrated MySQL database to store user information, bookings, and service details.
* **Responsive UI:** Designed with a clean, user-friendly interface that works across devices.

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML5, CSS3, JavaScript
* **Database:** MySQL
* **Tools:** Git, GitHub

## ⚙️ How to Get Started

Follow these steps to run the project locally on your machine:

### Prerequisites
* Python 3.x
* MySQL Server installed

### Installation
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Avnish2002/Machine_Wash_Management_System.git
    cd Machine_Wash_Management_System
    ```

2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Configure Database:**
    * Create a MySQL database.
    * Update your database connection settings in the project files if necessary.

4.  **Initialize the Database:**
    ```bash
    python init_db.py
    ```

5.  **Run the application:**
    ```bash
    python app.py
    ```
    The application will be accessible at `http://127.0.0.1:5000/`.
